# HOI4ABC
> Hearts of Iron 4 Achievments By Country.

## What this site is meant for
While I was achievment hunting I got really annoyed by having to scroll the Hoi4 Wiki looking for achievments for certain countries I wanted to play. So I made this to make a simple way to see every achievment each country has or can do.

All the information is pulled straight from the HOI4 Wiki, and I used AI to parse through it as I am too lazy to do that myself. So some things may be wrong if you notice any incorrect information open an issue and I will fix it.
